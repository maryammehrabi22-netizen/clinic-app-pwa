# سامانه مدیریت کلینیک 🏥

یک اپلیکیشن وب پیشرفته (PWA) برای مدیریت کلینیک پزشکی به زبان فارسی.

## امکانات

- 📊 **داشبورد** — آمار لحظه‌ای بیماران، نوبت‌ها و نسخه‌ها
- 👥 **مدیریت بیماران** — جستجوی زنده، اطلاعات بیمار و گروه خونی
- 📅 **نوبت‌دهی** — ثبت، فیلتر و مدیریت نوبت‌ها
- 💊 **نسخه‌ها** — لیست داروها و دوز مصرفی
- 📱 **PWA** — قابل نصب روی iPhone/iPad/Android

## نصب روی iPhone

1. در Safari آدرس را باز کنید
2. دکمه Share ↑ را لمس کنید
3. گزینه **Add to Home Screen** را انتخاب کنید
4. روی **Add** بزنید

## استفاده از GitHub Pages

در تنظیمات ریپو، بخش **Pages** را فعال کرده و branch `main` را انتخاب کنید.

## تکنولوژی

- HTML5 + CSS3 + Vanilla JavaScript
- فونت Vazirmatn
- Service Worker برای کارکرد آفلاین
- Safe Area برای notch و Dynamic Island
